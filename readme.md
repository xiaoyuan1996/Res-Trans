### Use This Framework

Step 1. 读data文件夹下的readme。

Step 2. 读取ExtractImage下的readme，并执行。

Step 3. 读取DataArgument下的readme，按步骤执行。

Step 4. 运行Train下的文件，运行多个模型生成不同的概率json。

Step 5. 运行Ensemble下的文件，进行模型集成，得到最后的提交文件。
